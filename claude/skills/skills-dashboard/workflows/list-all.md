# List All Skills

Lists all skills in `claude/skills/` and project-local `.claude/skills/` folders. Shows name, description, user-invocable status, and last-updated date for each skill.

- For each skill, display:
  - Name
  - Description
  - Path
  - user_invocable (if present)
  - Last updated (from file metadata)

**Next:**

- Type a skill name or number to see usage examples
- Type "search" to filter
- Type "skip" to exit
