# Create New Skill from Template

Launches the skill template generator workflow.

- Prompts user for skill name, description, and options
- Copies `claude/skills/skill-template/SKILL.md` to a new folder with the chosen name
- Optionally creates starter workflows

**Next:**

- Edit the generated skill file as needed
- Type "list" to return to all skills
- Type "skip" to exit

**Template location:**

- `claude/skills/skill-template/SKILL.md` (edit this to update the template for all new skills)
